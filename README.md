# NestJS Applicant Tracking System (ATS)

- **Developed** a RESTful backend using NestJS, TypeORM, and PostgreSQL, with JWT authentication and role-based access control.
- **Integrated** Google Gemini AI to parse data from uploaded resumes into user profiles and calculate compatibility scores between CVs and job descriptions.
- **Implemented** a candidate matching feature using SQL queries to rank applicants based on skill overlap, years of experience, and salary expectations.
- **Set up** real-time notifications using WebSockets (Socket.IO) to push application status updates to users.
- **Built** a subscription and credit tracking module, integrating VNPay to handle payment transactions.
- **Applied** database transactions to manage multi-table operations (e.g., user registrations, payment logging) to ensure data consistency.
- **Dockerized** the backend application to simplify local development setup and team collaboration.
